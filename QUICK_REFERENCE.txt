## Configuration Quick Reference

```toml
[Settings]
Max Iterations: 1-10  (default: 5)
  └─ Lower = faster, Higher = thorough

Model: 3 options
  ├─ gemini-2.5-flash  (fast)
  ├─ gemini-1.5-pro    (accurate)
  └─ gemini-1.5-flash  (balanced)

Temperature: 0.0-1.0  (default: 0.2)
  ├─ 0.0 = consistent
  ├─ 0.5 = balanced
  └─ 1.0 = creative
```

## Input Requirements

### Design Image
- ✅ **Formats**: PNG, JPG, JPEG, GIF, BMP
- ✅ **Size**: 800x600 or larger recommended
- ✅ **File Size**: < 20MB
- ✅ **Content**: Circuit diagrams, schematics, block diagrams

### Verilog Code
- ✅ **Formats**: Direct paste or .v, .sv, .verilog files
- ✅ **Size**: < 1MB code files
- ✅ **Content**: Valid RTL-level Verilog HDL

---

## Output Downloads

After verification, download:

```
1. Fixed Code
   ├─ Original + Fixed code comparison
   ├─ File: fixed_design.v
   └─ Format: Plain text Verilog

2. Detailed Report
   ├─ Executive summary
   ├─ Issue analysis
   ├─ Code quality assessment
   ├─ File: verification_report_YYYYMMDD_HHMMSS.md
   └─ Format: Markdown (.md)
```

## Troubleshooting Quick Fixes

```
Problem: "API Key not found"
Fix: Edit .env file with your key

Problem: App won't start
Fix: Check Python 3.8+, reinstall requirements.txt

Problem: Slow performance
Fix: Reduce max iterations, use faster model

Problem: Image won't upload
Fix: Check format (PNG/JPG), size < 20MB

Problem: API rate limit
Fix: Wait 30 seconds, upgrade API quota
```
